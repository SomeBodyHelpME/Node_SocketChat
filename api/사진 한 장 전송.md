## 사진 한 장 전송

| 메소드 | 경로          | 짧은 설명       |
| ------ | ------------- | --------------- |
| POST   | /photo/single | 사진 한 장 전송 |

### 요청 헤더

```json
Content-Type: multipart/form-data
```

### 요청 바디

```json
{
    "chatroom_idx" : 1,
    "u_idx" : 10,
    "photo" : 사진
}
```

* photo : 사진 1개

### 응답 바디

#### 컨텐츠 작성 성공

```json
{
    "status" : 201,
    "message" : "Success to Store Single Photo"
}
```

#### 사진이 없을 경우

```json
{
    "status": 400,
    "message": "No Photo"
}
```

#### 서버 내부 에러

```json
{
    "status": 500,
    "message": "Internal Server Error"
}
```
